## Упражнения

Позволь мне (<Latin text="acc" />.) теперь уйти. Возьми эти
драгоценные камни, которые я тебе дал (<Latin text="pt. pf. pass" />.).
Великий-поэт пусть сплетет венец-стихов из слов-жемчужин
(<Latin text="instr" />.). Ежедневно два вора обкрадывали
сокровищницу царя. Тот, кто от каждого (<Latin text="sarva" />)
принимает (<Latin text="p. pr. Ā" />.) дары, становится нечистым
(<Latin text="duṣ" />). Творец один сотворил мир своей волей. Ради
спасения иди (<Latin text="yā" />) под защиту богов. Цари пусть
удерживают злых путем наказания. Мы видели дочерей Рамы выходящими из
дома (<Latin text="pt. pr" />.). Жених должен перед огнем взять
руку девушки. Ариец не должен есть остатков (<Latin text="ucchiṣ" />,
<Latin text="pt. pf. pass" />.) другого. Следует ежедневно купаться
в текучей (не стоячей — <Latin text="nirudh" />) воде. Да защищает
вас трехглазый (бог), великий-господин (<Latin text="īśvara" />),
чело-которого украшено-серпом-луны!
